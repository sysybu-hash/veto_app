| [`01_QA_CHECKLIST.md`](./01_QA_CHECKLIST.md) | רשימת בדיקות מקיפה — 8-15 פריטים לכל מסך + E2E + ביצועים + אבטחה |

## איך להשתמש

1. **למעצב/לקוח** — פתח את `00_index.html` בדפדפן. עבור על כל המסכים, אשר/שנה.
2. **למפתח Flutter** — קרא את `00_SPEC.md` למסך → קרא את `11_DESIGN_SYSTEM.md` לטוקנים → ממש.
3. **ל-QA** — קרא את `01_QA_CHECKLIST.md`, הרץ את כל הפריטים לפני release.
